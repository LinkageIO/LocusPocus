Installation
------------
Option 1: Manually install the repo
```bash
$ git clone git@github.com:LinkageIO/LocusPocus.git && cd LocusPocus
$ python setup.py install
```
Option 2: Use pip
```bash
$ pip install git+https://github.com/schae234/LocusPocus.git
```
