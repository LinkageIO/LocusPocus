Installation
------------
Option 1: Use pip
```bash
$ pip install locuspocus
```

Option 2: Manually install the repo
```bash
$ git clone git@github.com:LinkageIO/LocusPocus.git
$ cd LocusPocus
$ pip install -e .
```
